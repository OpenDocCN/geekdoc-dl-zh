# 前言

![](img/356f7b2dad05044eec49e89d9ee8f6b0.png "《检索增强生成系统实用方法》")

在人工智能不断演变的领域中，检索增强生成（RAG）系统已经崛起为一个强大而多功能的工具。这些系统结合了信息检索和自然语言生成的优势，有潜力彻底改变我们访问、理解和交互大量数据的方式。随着我们开始阅读“**《检索增强生成系统实用方法》**”的篇章，我们深入探讨了这些系统开发和实施的核心原则、策略和技术。

本书是对 RAG 领域专家和爱好者所积累的知识、见解和经验的总结。我们将探讨结合检索和生成技术的复杂性，理解处理多样化数据源的细微差别，并见证查询路由、内存和可观察性工具的变革性影响。

在我们的探索核心，我们将利用“*与你的 PDF 文档聊天*”这一流行的真实世界用例来了解 RAG。无论你是经验丰富的 AI 从业者还是好奇的学习者，这些章节中的实用指导和动手示例旨在赋予你充分利用 RAG 系统全部潜力的能力。

一起，我们踏上掌握检索增强生成（Retrieval-Augmented Generation）艺术、发现解决复杂问题的创新方法，并最终改变我们与人工智能系统互动方式的旅程。加入我们的探索之旅，一起探索实用的 RAG 应用，让我们深入探索人工智能未来的变革之旅。
