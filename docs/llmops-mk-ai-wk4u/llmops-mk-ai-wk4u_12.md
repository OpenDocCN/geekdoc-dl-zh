# 2.1 简介

> 原文：[`boramorka.github.io/LLM-Book/en/CHAPTER-2/2.1%20Introduction/`](https://boramorka.github.io/LLM-Book/en/CHAPTER-2/2.1%20Introduction/)

LangChain 是一个开源框架，它将大型语言模型（LLMs），如 ChatGPT，连接到用户的内部和个人数据，使你能够“对话”文档，并从搜索引擎无法看到或模型训练后创建的内容中获得答案。由 Harrison Chase（LangChain 的联合创始人兼首席执行官）创建，这是让组织和个人真正使用他们自己的数据的关键一步。其核心思想是使信息访问民主化，并将“原始”数据转化为互动的、以对话驱动的知识来源：内部报告、研究、个人笔记——你现在可以像询问助手一样询问它们，无需 SQL 查询或手动文件搜索，从而加快分析并使数据工作更加高效。

LangChain 的架构是模块化的，旨在组装和部署 LLM 应用程序。其核心是提示，它为相关生成设置指令和上下文；模型，即理解上下文并产生类似人类答案的 LLMs；索引，它加快了索引和检索速度；链——多步骤处理管道，你可以在其中清理、分析和组合最终答案；以及代理——“指挥者”，它结合工具、管理数据流并适应特定任务的行为。这些元素共同构成了一个灵活的平台，可以定制以适应几乎任何数据景观和用例。

功能上，LangChain 覆盖了围绕 LLMs 的完整数据生命周期。它支持从多个来源和格式加载文档，并具有可配置的访问和密钥；提供预处理——将文本分割成具有语义意义的片段，以保留上下文并提高检索；通过嵌入和相似度度量实现语义搜索，以便你通过意义而不是关键词与数据交互；在对话场景中，提供“记忆”——跟踪先前消息并保持连贯对话，这自然地集成到链中。这种组合使 LangChain 成为助手、分析工具和运行在私有知识库上的企业机器人的绝佳选择。

要深入理解，请从官方文档和教程开始，依靠社区支持，并参加基本的 LangChain LLM-app 课程——这将使你快速掌握实践技能，并学会构建将内部数据与 LLMs 结合使用的解决方案。
