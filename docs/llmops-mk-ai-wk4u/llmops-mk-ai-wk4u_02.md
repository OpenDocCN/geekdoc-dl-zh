# 概述

> 原文：[`boramorka.github.io/LLM-Book/en/`](https://boramorka.github.io/LLM-Book/en/)

## 关于我

嗨！我是 Nikita Goryachev，Sber 的高级 AI/ML 工程师。我的团队在自然语言处理和推荐系统中实施最先进的算法。我们组织行业聚会，参加会议（新加坡的 RecSys，莫斯科的 AI Journey），并开发 RePlay——一个用于推荐的开源库。

## 关于本书

这本书是数据科学家、机器学习工程师、开发者和任何与现代人工智能工作的人的实用指南。我们探讨了大型语言模型（LLMs）、对话人工智能以及 LLMs 在开发流程中的集成，重点关注 LLMOps（大型语言模型 MLOps）。目标是提供清晰的工具和方法，以在实践中发挥人工智能的潜力。

方法和方法实例都是实用的：如何在商业场景中嵌入 LLMs（例如，使用聊天机器人进行客户支持、使用推荐进行个性化、使用 MLOps 提高效率）。本书有助于弥合复杂技术与实践之间的差距，展示如何使用人工智能实现增长和价值。

## 第一章：OpenAI API 基础

ChatGPT API 简介：功能、分类和应用。高级审查、增强机器推理、提示链、构建和评估 LLM 应用概述。

## 第二章：使用 LangChain 的对话聊天机器人

使用 LangChain 进行实用的聊天机器人开发：从环境设置到高级检索。特别关注上下文和对话记忆，以实现“类似人类”的交互。

## 第三章：LLMOps

结构化指南，介绍如何将 LLMs 集成到开发工作流程中：模型选择和调整、部署、监控、自动化和最佳实践。实际案例和伦理方面。

这本书不仅仅是一系列技术的集合，而是关于负责任和创新人工智能使用的全面指南。在这里，你可以找到技术、伦理和实践方面，以及 LLMOps 快速变化世界中发展的路线图。
