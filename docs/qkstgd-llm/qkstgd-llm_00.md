## 前言

近年来，大型语言模型（LLMs）的进步彻底改变了自然语言处理领域。BERT、T5 和 ChatGPT 等模型在从文本分类到机器翻译的广泛 NLP 任务上展现了前所未有的性能。尽管它们的性能令人印象深刻，但 LLMs 的使用对于许多从业者来说仍然具有挑战性。这些模型的大小以及对其内部工作原理缺乏理解，使得从业者难以有效地使用和优化这些模型以满足其特定需求。

这本关于在自然语言处理中使用 LLMs 的实用指南概述了 LLMs 中使用的核心概念和技术，并解释了这些模型的工作原理以及如何用于各种 NLP 任务。本书还涵盖了高级主题，如微调、对齐和信息检索，同时提供了针对特定 NLP 任务的 LLMs 训练和优化的实用技巧和窍门。

本工作涵盖了大型语言模型（LLMs）领域广泛的主题，包括 LLMs 的基础知识、使用专有模型启动应用程序、使用自定义示例微调 GPT3、提示工程、构建推荐引擎、结合 Transformers 以及将自定义 LLMs 部署到云端。它深入探讨了大型语言模型领域使用的各种概念、技术和工具。

涵盖的主题：

1. 使用大型语言模型（LLMs）进行编码

2. 使用专有模型的概述

3. OpenAI、嵌入、GPT3 和 ChatGPT

4. 向量数据库和构建神经/语义信息检索系统

5. 使用自定义示例微调 GPT3

6. 使用 GPT3 和 ChatGPT 进行提示工程

7. 高级提示工程技术

8. 构建推荐引擎

9. 结合 Transformers

10. 将自定义 LLMs 部署到云端
