# 什么是图层规范化？

> 原文：<https://www.assemblyai.com/blog/what-is-layer-normalization/>

您可能以前听说过批处理规范化。这是一个伟大的方式，使您的网络更快，更好，但有一些缺点批量规范化。这就是为什么研究人员提出了一种对批处理规范化的改进，称为层规范化。

在本视频中，我们将了解图层规范化的工作原理、它与批量规范化的比较以及它在什么情况下最有效。
