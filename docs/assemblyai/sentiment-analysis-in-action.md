# 行动收益电话会议中的情绪分析

> 原文：<https://www.assemblyai.com/blog/sentiment-analysis-in-action/>

转录音频很酷，但如何从非结构化数据中获得洞察力呢？有了 AssemblyAI，你可以从你的音频或视频文件中得到更多。

在这个视频中，我们制作了一个网络应用程序，对公司的收益电话会议进行情绪分析。只需输入盈利电话的录音，就能从电话中了解你的观点。大家来学习一下怎么做吧！
