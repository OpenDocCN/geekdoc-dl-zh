# Haskell.org 委员会：ezyang 的博客

> 来源：[`blog.ezyang.com/2010/11/haskell-org-committee/`](http://blog.ezyang.com/2010/11/haskell-org-committee/)

## Haskell.org 委员会

试过访问 haskell.org 发现它宕机了吗？现在你有人可以抱怨了：[haskell.org 委员会](http://haskellorg.wordpress.com/2010/11/15/the-haskell-org-committee-has-formed/)已经成立，而我显然是其中一员。`8-)`

我们将首先要做的事情之一是将 haskell.org 从目前托管在耶鲁大学的服务器（目前的托管效果不错，但周末服务器将会宕机，直到周一才有人去启动它）迁移到一些[专用硬件](http://new-www.haskell.org/haskellwiki/Haskell)上。我必须承认，我作为委员会的一员却没有任何经验（或直接经验）来帮助进行维护工作，对此我感到有些不好意思，希望这种情况会很快改变。
