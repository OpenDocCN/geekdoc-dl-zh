# 前言

这是一本关于 `torch` 的书，它是 PyTorch 的 R 接口。截至本文写作时，PyTorch 是主要的深度学习和科学计算框架之一，在各个行业和研究领域得到广泛应用。使用 `torch`，你可以直接从 R 访问其丰富的功能，无需安装，更不用说学习 Python。尽管作为一个项目仍然“年轻”，`torch` 已经拥有一个充满活力的用户和开发者社区；后者不仅扩展了核心框架，还在自己的包中基于它进行构建。

在本文中，我试图实现三个目标，对应于本书的三个主要部分。

第一部分是对核心 `torch` 的全面介绍：没有这些基本结构，任何东西都无法工作。尽管在未来的工作中，你可能会尽可能使用高级语法结构，但了解它们所负责的内容以及理解核心概念是非常重要的。更重要的是，从实际的角度来看，你只需要在某种程度上对 `torch` 感到“流利”，因此你不必经常求助于“试错编程”。

在第二部分，基础知识讲解之后，我们继续探索深度学习的各种应用，从图像识别到时间序列和表格数据，再到音频分类。在这里，重点也是概念解释。此外，每一章都提供了一个你可以用作自己应用“模板”的方法。在适当的时候，我也试图指出结合领域知识的重要性，而不是不常见的“大数据、大模型、大计算”方法。

第三部分特别之处在于它突出了你可以使用 `torch` 做的一些非深度学习的事情：矩阵计算（例如，解决线性回归问题的各种方法）、计算离散傅里叶变换和小波分析。在这里，比其他任何地方都更加重要的是概念方法。让我来解释一下。

首先，我预计在教育背景方面，我的读者会有很大的差异。随着 R 语言在自然科学以及其他接近应用数学的领域的教学和使用的日益增加，有些人可能会觉得他们从对诸如离散傅里叶变换如何工作的概念性（尽管公式引导！）解释中获益不多。然而，对另一些人来说，这些内容可能是一片未知的领域，如果一切按常规进行，他们可能永远不会涉足。例如，对于具有人文主义背景、非传统经验主义导向的人，如文学、文化研究或语言学的人来说，情况可能就是这样。当然，如果你属于后者，你可能会发现我的解释，尽管以概念为中心，但仍然非常（或：过于）数学化。在这种情况下，请放心，对于理解这些事物（以及许多其他值得理解的事物）来说，这是一条漫长的道路；但我们有终身的时间。

其次，尽管深度学习在过去十年中一直是“主流”范式，但最近的发展似乎表明，对数学/领域基础的兴趣（*再次* – 这是一个反复出现的现象）正在上升（例如，考虑 Bronstein 等人系统解释的几何深度学习方法(2021)，以及[*超越炼金术：几何深度学习初探*](https://blogs.rstudio.com/ai/posts/2021-08-26-geometric-deep-learning/)中的概念介绍。）在未来，我假设我们将越来越可能看到更多将深度学习技术和领域知识相结合的“混合”方法。傅里叶变换不会消失。

最后但同样重要的是，关于这个话题，让我明确指出，当然，所有章节都有`torch`代码。例如，对于傅里叶变换，你不仅会看到使用专用功能进行的官方方法，还会看到自己编写算法的各种方式 – 在令人惊讶的少量行数中，并且具有非常令人印象深刻的表现。
